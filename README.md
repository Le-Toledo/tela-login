# 🔐 Sistema de Login com Flask

Projeto simples de tela de login e cadastro usando Python e Flask.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Login de usuários
- Senhas protegidas com hash (`werkzeug.security`)
- Banco de dados SQLite
- Interface simples com HTML + CSS

---

## 🛠️ Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML / CSS
- Werkzeug
